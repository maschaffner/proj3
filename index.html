Discover Boston was designed and built by Alain Ibrahim and Mike Schaffner

The four main parts of this dashboard-style visualization are:

  1. The Overhead Map, which allows the user to efficiently compare different zip codes' locations and general measures such as number of homes for sale (or sold), as well as average price of those homes. It also allows a user to click a specific zip code bubble for more details (see #3) and to reveal those listings on the scatterplot.
  2. The Price-per-SqFt Scatterplot shows all of the listings in the area (filtered by the Controls), and is also linked to the Overhead Map.
  3. The Zip Code Bullet Graph appears upon selecting a zip code bubble, and shows the average price, Zestimate price, and max price of the selected zip code to better inform the viewer of the properties of that zip code.
  4. The Controls, which are linked to all of the other views of the visualization.

Each of these enables the general principles of interactive visualizations:

  1. Overhead View First
  2. Zoom and Filter
  3. Details on Demand
